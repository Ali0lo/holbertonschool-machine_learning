# Neural Style Transfer

This repository contains implementations of the Neural Style Transfer (NST) class using TensorFlow and VGG19.

Files:
- 0-neural_style.py → Image preprocessing and scaling
- 1-neural_style.py → Model loading
- 2-neural_style.py → Gram matrix calculation
- 3-neural_style.py → Feature extraction
- 4-neural_style.py → Layer style cost
- 5-neural_style.py → Total style cost
- 6-neural_style.py → Content cost
- 7-neural_style.py → Total cost
- 8-neural_style.py → Gradient computation
- 9-neural_style.py → Image generation
- 10-neural_style.py → Variational cost support

The implementation follows the paper:
"A Neural Algorithm of Artistic Style"
by Leon A. Gatys et al.
