# Clustering Algorithms

This repository contains implementations of clustering algorithms and Gaussian Mixture Models.

Files:
- 0-initialize.py → K-means centroid initialization
- 1-kmeans.py → K-means clustering
- 2-variance.py → Intra-cluster variance
- 3-optimum.py → Optimum k by variance
- 4-initialize.py → GMM initialization
- 5-pdf.py → Gaussian PDF
- 6-expectation.py → Expectation step
- 7-maximization.py → Maximization step
- 8-EM.py → Expectation Maximization
- 9-BIC.py → Bayesian Information Criterion
- 10-kmeans.py → sklearn KMeans
- 11-gmm.py → sklearn GaussianMixture
- 12-agglomerative.py → Agglomerative clustering
