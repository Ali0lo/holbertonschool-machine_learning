# Autoencoders

## Files

- **0-vanilla.py** — Vanilla autoencoder with dense encoder/decoder and binary crossentropy loss.
- **1-sparse.py** — Sparse autoencoder with L1 activity regularization on the latent layer.
- **2-convolutional.py** — Convolutional autoencoder using Conv2D, MaxPooling2D, and UpSampling2D.
- **3-variational.py** — Variational autoencoder (VAE) with reparameterization trick and KL divergence loss.
